# Fashion Fiesta - Operations Guide

This file contains the commands to run and manage your new Next.js + FastAPI stack.

## üöÄ How to Run the Application

### 1. Backend (FastAPI)
Open a terminal in the `backend` folder:
```bash
# Navigate to backend
cd backend

# Install dependencies (First time only)
pip install -r requirements.txt

# Start the server (Dev mode)
uvicorn main:app --reload
```
*API will be available at: http://localhost:8000*
*Interactive Documentation (Swagger): http://localhost:8000/docs*

### 2. Frontend (Next.js)
Open a NEW terminal in the `frontend` folder:
```bash
# Navigate to frontend
cd frontend

# Install dependencies (First time only)
npm install

# Start the development server
npm run dev
```
*WebApp will be available at: http://localhost:3000*

## üõ†Ô∏è Project Management Commands

### Database Migrations (Backend)
If you update the models in `models.py`, use Alembic for migrations (recommended for scale):
```bash
alembic revision --autogenerate -m "description"
alembic upgrade head
```

### Frontend Build
To create a production-ready build:
```bash
cd frontend
npm run build
npm run start
```

### Environment Variables
- Backend: Create a `.env` file in `backend/` to specify `DATABASE_URL`.
- Frontend: Create a `.env.local` file in `frontend/` for `NEXT_PUBLIC_API_URL`.
